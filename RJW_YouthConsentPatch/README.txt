RJW Youth Consent Patch
========================

Author: NotYourBusiness

Description:
This small patch allows Youth and Adult pawns to have sexual relationships with a configurable minimum age.

Features:
- Slider in Mod Settings (range 13-17)
- Default minimum age: 16
- Works both ways (either pawn can initiate)
- Toggle the feature on/off

Installation:
1. Drop the folder into your Mods directory
2. Make sure it loads AFTER RimJobWorld
3. Configure the minimum age in Mod Settings

Compatible with latest RJW (rjw-master).

Have fun and play responsibly!
